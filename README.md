# Portfolio - Mohammad Jafar Saberi

![Preview](./public/portfolio.png)

Modern developer portfolio built with React, TypeScript, and Tailwind CSS.

## Features

- Responsive design
- Project showcase
- Downloadable CV

## Tech

- React + TypeScript
- Tailwind CSS
- Vite

## Setup

```bash
git clone https://github.com/mohammadjafarsaberi/portfolio.git
cd portfolio
npm install
npm run dev
```
